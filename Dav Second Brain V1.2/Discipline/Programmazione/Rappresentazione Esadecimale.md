#uni 
Associamo ad un simbolo esadecimale una stringa di 4 bit.
In informatica questa rappresentazione viene indicata dal prefisso $0x$
Es:
$0xC1 \implies 1100 | 0001$

| 0010 | 2   |     | 1100 | C   |
| 0011 | 3   |     | 1101 | D   |
| 0100 | 4   |     | 1110 | E   |
| 0101 | 5   |     | 1111 | F   |
| 0110 | 6   |     |      |     |
