Dati $x^1,...,x^k \in R^n$
Un punto $y$ si dice combinazione conica di $x^1,...,x^k$ se $\exists \lambda _i \geq 0, i = 1,...,k$ 
tale che $$y = \sum_{i=1} ^k \lambda _ix^i$$
L'insieme di tutte le possibili combinazioni coniche

```functionplot
---
title: Combinazione Conica
xLabel: x
yLabel: y
bounds: [0,10,0,10]
disableZoom: false
grid: true
---
y(x) = 2x
y2(x) = x
```
